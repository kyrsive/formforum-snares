# formforum-snares

a collection of experimental snare sounds to be used on strudel from various sound designers under the label [formforum](https://formforum.bandcamp.com/) 

note that some snares can get extremely weird (can be extremely long, loud, or literally nothing) so be careful live coding with this (using the cut parameter can help)

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/formforum-snares')
```
